## Build Your Year

an interactive goal-setting experience that transforms your ambitions into a playable bingo board. Instead of writing a traditional resolution list, users draw their goals directly into each square and watch their board come to life.
As goals are completed, stamps appear on the board—and when a row, column, or diagonal is filled, the site celebrates with a satisfying BINGO moment, giving users a sense of progress and motivation to keep going.

The board automatically updates to the current year, making it a timeless tool you can return to again and again.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
